# NSUT Alumni Network

NSUT Alumni Network is a platform built to connect students and alumni of Netaji Subhas University of Technology through a unified, structured system.

## What it does

- Alumni directory with searchable profiles  

- Student–alumni interaction and outreach  

- Opportunities, mentorship, and referrals  

- Event and community management  

- Centralized alumni data and records  

## Why it exists

Alumni ecosystems are typically fragmented and underutilized. This platform creates a single system to enable meaningful connections, improve access to opportunities, and strengthen the university network.
